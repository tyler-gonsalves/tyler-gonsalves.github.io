Capstone Design Project: Digital Twin


Video Coming Soon

The Digital Twin Project encorporates embedded system design and machine learning into one project. Using sensors to analyze the current state of a physical system, we can use machine learning to predict the remaining useful life of the system. We can take the project even further by sending signals to the system to tweak and optimize its performance to maximize the remaining life of the system, saving time and money on repairs and maintenance.

Embedded Systems Design Final Project: Interactive Weather Application


Video Coming Soon

The Interactive Weather Application utilizes a DE1-SOC FPGA, Embedded C, and Python to display weather information using coordinates specified by the user. The user uses a joystick to move crosshairs around a weather map of the United States, and the application will send accurate weather information to the users screen.
